# Happy Chatting

Happy chatting is a real time chat app with websockets using express and socket.io Vanilla JS on the frontend.

## Stack

### Front-End

[1]. HTML.\
[2]. CSS.\
[3]. Vanilla JavaScript.\

### Back-End

[1]. Express.\
visit [Express]([https://expressjs.com/)\
[2]. Socket-io.\
visit [Socker-io](https://socket.io/)\

## Usage

```
    npm install
    npm run dev

```
